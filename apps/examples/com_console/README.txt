   o STM32 or MBHP_CORE_STM32

===============================================================================

This application provides a tiny console for sending commands to the application
via COM interface

Use a terminal program like Hyperterm (Windows), Minicom (Linux) or ZTerm (MacOS)
to open a USB COM connection to the core.
